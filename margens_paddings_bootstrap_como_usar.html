Margens e Paddings no Bootstrap 5

Classes de Margem
m-0

Aplica nenhuma margem em todos os lados do elemento.

m-1

Aplica uma margem pequena em todos os lados.

m-2

Aplica uma margem média pequena em todos os lados.

m-3

Aplica uma margem média em todos os lados.

m-4

Aplica uma margem grande em todos os lados.

m-5

Aplica uma margem muito grande em todos os lados.

m-auto

Aplica margem automática, geralmente usada para centralizar horizontalmente.

Margens por Direção
mt-3

Aplica uma margem de tamanho 3 no topo.

mr-3

Aplica uma margem de tamanho 3 na direita.

mb-3

Aplica uma margem de tamanho 3 no fundo.

ml-3

Aplica uma margem de tamanho 3 na esquerda.

mx-3

Aplica uma margem de tamanho 3 nas laterais (esquerda e direita).

my-3

Aplica uma margem de tamanho 3 no topo e no fundo.

Classes de Padding
p-0

Aplica nenhum padding em todos os lados do elemento.

p-1

Aplica um padding pequeno em todos os lados.

p-2

Aplica um padding médio pequeno em todos os lados.

p-3

Aplica um padding médio em todos os lados.

p-4

Aplica um padding grande em todos os lados.

p-5

Aplica um padding muito grande em todos os lados.

Padding por Direção
pt-3

Aplica um padding de tamanho 3 no topo.

pr-3

Aplica um padding de tamanho 3 na direita.

pb-3

Aplica um padding de tamanho 3 no fundo.

pl-3

Aplica um padding de tamanho 3 na esquerda.

px-3

Aplica um padding de tamanho 3 nas laterais (esquerda e direita).

py-3

Aplica um padding de tamanho 3 no topo e no fundo.
