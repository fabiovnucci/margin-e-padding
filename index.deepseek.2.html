Abreviações de Margin e Padding no Bootstrap 5

Visualize e entenda como funcionam as classes utilitárias de espaçamento

Como funcionam as abreviações

  • Propriedade: m = margin, p = padding
  • Lados:
    • t = top (topo)
    • b = bottom (base)
    • s = start (início, esquerda em LTR)
    • e = end (fim, direita em LTR)
    • x = eixo horizontal (esquerda + direita)
    • y = eixo vertical (topo + base)
  • Tamanho: 0 a 5
    • 0 = 0
    • 1 = 0.25rem (4px se root for 16px)
    • 2 = 0.5rem (8px)
    • 3 = 1rem (16px)
    • 4 = 1.5rem (24px)
    • 5 = 3rem (48px)
  • Exemplo: mt-3 = margin-top: 1rem

Exemplos de Código

<!-- Exemplo de uso das classes -->
<div class="p-2">
  <div class="m-2">
    Conteúdo com espaçamento
  </div>
</div>
/* Equivalente em CSS */
.p-2 {
  padding: 0.5rem;
}

.m-2 {
  margin: 0.5rem;
}
Abreviação Propriedade CSS Descrição
m ou p margin ou padding Aplica em todos os lados do elemento
mt ou pt margin-top ou padding-top Aplica apenas no topo do elemento
mb ou pb margin-bottom ou padding-bottom Aplica apenas na base do elemento
ms ou ps margin-start ou padding-start Aplica no início (esquerda em layouts LTR)
me ou pe margin-end ou padding-end Aplica no fim (direita em layouts LTR)
mx ou px margin-left/right ou padding-left/right Aplica nos lados horizontais
my ou py margin-top/bottom ou padding-top/bottom Aplica nos lados verticais
